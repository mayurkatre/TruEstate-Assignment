# TruEstate R.S.M.S. - Sales Management System (Backend)

This is the backend component of the TruEstate R.S.M.S. Sales Management System.

## Structure

- `dataset/` - Database migrations and initial data
- `src/` - Source code
  - `controllers/` - Request handlers
  - `services/` - Business logic
  - `models/` - Data models and types
  - `routes/` - API route definitions
  - `utils/` - Utility functions
  - `index.js` - Entry point

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

## Deployment

```
npm start
```
